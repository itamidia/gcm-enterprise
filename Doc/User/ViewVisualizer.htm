Visualização de consulta

Uma consulta é uma forma de visualizar registros de diversos arquivos de uma só vez obedecendo a um critério especial. Após prepararmos uma determinada consulta podemos gravá-la para visualizar mais tarde.

As instruções abaixo devem ser seguidas na visualização de consultas. A tela abaixo é um exemplo de como poderá aparecer o menu de consultas disponíveis para seleção. Qualquer consulta pode ser selecionada a cada vez. Se nenhuma consulta tiver sido criada, a lista estará vazia e só receberá as consultas que forem elaboradas.
 

Eliminando uma consulta existente

Para eliminar uma consulta já gravada, basta selecionar a consulta desejada e clicar sobre o botão excluir. Desta forma, a consulta não aparecerá mais na lista.

Visualizando uma consulta já existente

Podemos selecioná-la e clicar sobre o primeiro botão. Ao abrir a visualização, podemos clicar no botão de consulta e estabelecer condição de seleção e pesquisa de registros (veja tópico Filtro de dados);

Opções de consulta

Durante a visualização da consulta pode-se utilizar o botão direito do mouse para ter acesso a um menu de atalho(pop-up) que oferece as seguintes operações:

  • Apagar um coluna selecionada para que não seja visível na consulta;
  • Filtrar os dados visualizados criando um novo conjunto de registros que podem ser visualizados;
  • Criar um gráfico das colunas selecionadas;
  • Imprimir a consulta em visualização;
  • Procurar para posicionar o cursor em um determinado registro da consulta;
  • Reconstruir a consulta com o seu formato original;
  • Totalizar um coluna numérica.
