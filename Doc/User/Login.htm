Acessando a aplicação

Toda vez que a aplicação GCM Enterprise - Sistema Admin é ativada, é apresentada uma tela solicitando o nome do usuário e a senha para acessar a aplicação.

Porém, na primeira vez que a aplicação é executada, o usuário padrão ADMIN e a senha padrão CNME122518 são criados automaticamente e possuem livre acesso a todas as opções que a aplicação oferece. Devemos salientar, no entanto, que tanto o usuário quanto a senha são informações que precisam estar corretas, caso contrário o acesso à operação da aplicação será negado. Após esse primeiro acesso, o usuário poderá criar um nome de usuário e a senha que desejar, bem como atribuir as permissões devidas a cada uma das opções da aplicação.
