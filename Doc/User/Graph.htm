Gráfico

Durante a visualização de uma consulta ou trabalhando no formato grade, podemos selecionar colunas e criar gráficos para serem visualizados, impressos ou gravados em disco, para serem usados mais tarde em outras aplicações.Os gráficos representam os dados que estão na grade. Conforme alteramos os dados da grade, o gráfico também é alterado.

Para traçar um gráfico do preço de custo versus o preço indexado, siga os passos seguintes:

  1. Selecione a coluna Código, clicando com o mouse sobre área do título da coluna;
  2. Clique em seguida com o botão direito do mouse para ter acesso ao menu de atalho e escolha a opção Apagar colunas para apagar a coluna selecionada.
  3. Selecione as colunas Descrição, Preço de custo e Preço indexado segurando o mouse na região do título da coluna Descrição e arrastando-o para até a coluna Preço indexado;

Com isto, a consulta terá a seguinte aparência:

Após selecionadas as colunas, clique o botão direito do mouse para acessar o menu de atalho que apresentará as opções disponíveis. Então, clique sobre a opção Grafar.

Uma maneira mais rápida e prática é clicar diretamente sobre o botão que se encontra na barra de ferramentas

Alterando a forma de visualização do Gráfico

Com o mouse posicionado em qualquer região do gráfico, clique com o botão direito e será apresentada uma tela onde você poderá alterar a forma de visualizar o gráfico.

Pode-se alterar todos os dados que se desejar, como por exemplo o tipo do gráfico, o título do gráfico e o que mais for necessário.

Para confirmar as informações escolhidas, OK

Para restaurar as configurações do gráfico original clique sobre o botão "restaurar"

Para cancelar as operações de configuração do gráfico, clique sobre o botão "cancelar"

Gravando um gráfico

Para gravar a imagem de um gráfico exportando-a para um arquivo de formato .BMP ou .WMF, para futura visualização com outro aplicativo, quando um gráfico estiver sendo apresentado, selecione a opção Gravar, a partir do menu ou clique sobre o botão , na barra de ferramentas

Imprimindo a imagem de um gráfico

Para imprimir a imagem de um gráfico enviando-a para a impressora configurada como padrão no Windows, selecione a opção Imprimir a partir do menu ou clique sobre o botão , na barra de ferramentas
