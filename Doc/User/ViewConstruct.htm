Criação de consulta

Uma consulta é necessária quando precisamos de uma seleção de informações em uma pesquisa. Os dados pesquisados na consulta obedecem a parâmetros determinados pelos próprios usuários.

Selecione o módulo Construtor de Consulta para criação de uma nova consulta

Preencha os campos de acordo com as explicações abaixo:.

Nome da Consulta

Insira neste campo o nome pelo qual a consulta será chamada, durante a execução da aplicação.

Campos a serem apresentados

Selecione as tabelas e os campos desejados onde será feita a consulta. Vários campos podem ser escolhidos.

Agrupar por

È um recurso muito utilizado na classificação ou agrupamento de informações que possuem pontos em comum, por exemplo, clientes de uma mesma cidade ou funcionários de uma mesma categoria.

Ordenação

A ordenação dos dados que serão mostrados pode ser de forma ascendente ou descendente. Selecione a opção desejada na seção de ordenação. Caso vários campos tenham sido escolhidos na consulta, um dos campos poderá ser escolhido para receber a ordenação, nesse caso, os outros campos obedecerão a ordem do campo escolhido. Observe os campos disponíveis e selecione aqueles que preferir.

Os Primeiros

Se desejar que somente os primeiros registros de uma mesma categoria sejam exibidos, para facilitar a visualização, utilize a opção “Os primeiros”, digitando o número desejado dos registros que serão exibidos primeiro.

Seleção de registros/união de tabelas

Se precisar selecionar registros específicos, a opção Criar Filtragem satisfaz essa necessidade. A opção Unir Tabelas pode ser utilizada para agregar os dados de outra tabela àquela já em uso. Os parâmetros abaixo são necessários quando uma filtragem é usada:

    Nome do campo = Insira o nome do campo cujos dados serão filtrados.

    Operadores = Utilize um dos operadores disponíveis para definir a condição de filtragem.

    Valor = é o parâmetro principal da filtragem, o conteúdo do campo deverá ser o mesmo que esse valor.

Criar Filtragem

Utilize esta opção se preferir fazer uma seleção dos registros em pesquisa.

União de Tabelas

A opção Unir Tabelas deve ser utilizada sempre que se desejar comparar campos de duas tabelas, neste caso o campo Valor será transformado em uma lista a partir da qual pode-se escolher qual campo de qual tabela será comparado. A união de tabelas irá considerar registros das duas tabelas que tenham conteúdos iguais nos campos informados, estabelecendo a cláusula WHERE (ver glossário).

Para estabelecer uma condição siga os passos abaixo:

  • Clique no campo Nome do Campo para que seja mostrada uma lista com os nomes dos campos que fazem parte da consulta, escolha o campo desejado para estabelecer uma condição;
  • Clique no campo Operador e escolha um dos operadores constantes da tabela a seguir, que corresponde às opções disponíveis na lista:
Tecla Função que desempenha
=igual a...
<>diferente de...
>maior que...
>=maior ou igual a
<menor que...
<=menor ou igual a...
<menor que...
LIKEoperador especial que usa caracteres
coringa (veja este termo no Glossário)

Clique no campo Valor e insira a informação que deverá ser comparada com o valor contido na campo escolhido (não use aspas para valores literais constantes).

Neste ponto foi estabelecida apenas uma condição para a consulta. Agora, clique sobre o botão Concatena para aceitar esta primeira condição. Se desejar estabelecer outras condições para serem concatenadas a esta primeira, clique sobre uma das opções de conexão E / OU e proceda conforme descrito anteriormente para o primeiro critério.

As informações que vão sendo mostradas na janela de criação de consultas são instruções de uma linguagem denominada SQL. Não é necessário conhecer esta linguagem pois diversos termos podem ser consultados no glossário. O sistema GCM Enterprise - Sistema Admin monta automaticamente esta expressão e a mostra apenas para servir de referência para o usuário saber quais tabelas e campos já selecionou para participar da consulta, bem como as ordenações e filtragens que tenha feito.

Para limpar todas as informações digitadas para uma consulta, permitindo que uma nova consulta seja elaborada, clique sobre o texto e apague seu conteúdo
